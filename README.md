## 贡献流程
1. Fork本项目到你自己的账号下，并clone到本地
1. 查看『章节认领』后提交一个Issue，申请你要翻译的章节。如：申请翻译『AdonisJs一览』
1. 得到回复后即可在你本地翻译对应章节，为保证效率，强烈建议配合 [Google Translate](https://translate.google.cn/)
1. 请务必于3日内完成翻译，完成后发送Pull Request
